Inside Jotform builder
Apply the theme to each form
Do these steps once — then repeat for all 4 forms. Takes about 10 minutes per form. Start with the Registration form.
Part 1 — Open the Form Builder
1
Log in to jotform.com → click My Forms → find "Identity and Purpose Group Coaching – Registration" → click Edit.
2
In the form builder, click the Paint Brush icon on the left sidebar — this opens the Form Designer panel.
Part 2 — Apply Thrive Colours
3
In Form Designer, click "Customize Theme" or scroll to find individual colour pickers. Set Form Background to #FDF9F3.
4
Set Button Color to #C9963A. Set button text to #FFFFFF. Change button label to "Reserve My Spot →".
5
Set Selected / Active element colour (radio buttons, checkboxes tick) to #C9963A.
6
Set Input border focus colour to #C9963A. Set label text to #1A3028.
Part 3 — Style the Header
7
Click the Header section at the top of your form. Set background colour to #1A3028. Set header text colour to #FAF6EE.
8
Update the header Title to match the form's purpose (see "All 4 Forms" tab for exact titles). Add a short subtitle in the description field.
9
Add a Scripture verse as a Text element just below the header — using the gold border-left style via Custom CSS (see CSS tab). Each form has its own verse.
Part 4 — Set Typography
10
In Form Designer → Font section: set Question Font to DM Sans. If DM Sans isn't available in Jotform's list, choose Lato as the closest match.
11
Set question label font size to 14px, weight to Medium (500). Set input text size to 14px, weight Regular (400).
Part 5 — Save and Repeat
12
Click Save in the top right. Preview the form on mobile. Then go back to My Forms and repeat steps 1–11 for the remaining 3 forms.
13
Pro move: After styling the first form, Jotform lets you save it as a Theme. Click Save as Theme in the designer panel, name it "Thrive Brand", then apply it to all other forms in one click.
Advanced · Custom CSS
Pixel-perfect Thrive styling
Paste this into Jotform's Custom CSS box (Form Designer → Advanced → Custom CSS) to get exact landing page styling that Jotform's colour pickers alone can't achieve.
How to add Custom CSS in Jotform
Form Builder → Paint Brush icon → scroll to the bottom → "Advanced" section → "Custom CSS" text area → paste the code below → Save.
/* ── Thrive Brand CSS for Jotform ── */ /* Page background */ .jotform-form { background-color: #FDF9F3 !important; font-family: 'DM Sans', 'Lato', sans-serif !important; } /* Form header */ .form-header-group { background: #1A3028 !important; border-radius: 14px 14px 0 0 !important; padding: 24px 28px !important; } /* Header title */ .form-header-group .form-header { color: #FAF6EE !important; font-family: 'Playfair Display', Georgia, serif !important; font-weight: 900 !important; font-size: 24px !important; } /* Header subtitle */ .form-header-group .form-subHeader { color: rgba(250,246,238,0.65) !important; font-weight: 300 !important; } /* Question labels */ .form-label { color: #1A3028 !important; font-weight: 500 !important; font-size: 14px !important; } /* Input fields */ .form-input, .form-textarea, .form-dropdown { border: 1px solid rgba(26,48,40,0.15) !important; border-radius: 8px !important; background: #FFFFFF !important; color: #1A3028 !important; font-size: 14px !important; padding: 10px 14px !important; } /* Input focus ring — gold */ .form-input:focus, .form-textarea:focus { border-color: #C9963A !important; box-shadow: 0 0 0 3px rgba(201,150,58,0.12) !important; outline: none !important; } /* Radio / checkbox accent */ .form-radio-item input:checked + label::before, .form-checkbox-item input:checked + label::before { background-color: #C9963A !important; border-color: #C9963A !important; } /* Submit button */ .form-submit-button, button[type="submit"] { background: #C9963A !important; color: #FFFFFF !important; border: none !important; border-radius: 99px !important; font-weight: 500 !important; letter-spacing: 0.02em !important; padding: 14px 32px !important; transition: background 0.2s !important; } .form-submit-button:hover { background: #b0812e !important; } /* Scripture verse block (add as Text element) */ .supernova-form .form-section-content p { border-left: 2px solid #F0C97A !important; padding-left: 12px !important; font-style: italic !important; color: #C9963A !important; font-size: 13px !important; line-height: 1.6 !important; }
Paste this CSS once into your first form, save it as a Jotform Theme named "Thrive Brand", then apply that saved theme to the remaining 3 forms — no need to re-paste the CSS each time.
